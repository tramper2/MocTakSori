# ☸️ 목탁 수행 (MocTakSori) - 마음의 평온

바쁜 현대 사회에서 마음속 번뇌와 스트레스를 내려놓을 수 있도록 돕는 **반응형 목탁 명상 웹 애플리케이션**입니다.  
PC와 모바일 기기 모두에 대응하며, 터치와 클릭 시 즉각적이고 사실적인 목탁 소리 및 영롱한 비주얼 피드백을 전달하여 몰입도 높은 명상 수행을 제공합니다.

---

## 🌟 주요 기능

1. **맑은 목탁 사운드 합성 (Web Audio API)**
   * 오디오 파일을 불러오지 않고 브라우저 자체적으로 목탁 음향을 실시간 합성하여 레이턴시(지연 시간)가 전혀 없는 즉각적인 타격 반응을 실현했습니다.
   * 실제 목탁의 물리 구조를 본뜬 하모닉스 배음(배음 주파수 비율 1.58배) 및 어택 노이즈, 피치 스윕 효과를 주었습니다.
   * 타격할 때마다 주파수가 ±12Hz 내에서 불규칙적으로 변조되어, 사람이 직접 치는 듯한 자연스러운 소리를 냅니다.
   * 수행 설정에서 목탁의 기본 주파수(음높이)와 볼륨을 취향에 맞춰 자유롭게 세팅할 수 있습니다.

2. **반응형 오디오 페이드 인/아웃 상태 머신**
   * 목탁을 칠 때 선택한 불경(반야심경, 천수경, 관세음보살정근)이 낭독됩니다.
   * 사용자가 설정한 '자동 멈춤 시간(기본 8초)' 동안 아무런 클릭/터치 입력이 없으면 불경 낭독 소리가 부드럽게 페이드 아웃되며 정지합니다.
   * 다시 목탁을 치면 소리가 부드럽게 페이드 인 되며 정지했던 위치부터 자연스럽게 이어서 낭독을 시작합니다.

3. **3가지 오디오 독경 모드**
   * **차분한 내장 독송 (TTS):** 기기 내장 음성 엔진을 사용하여 명상에 어울리는 느린 박자(0.78배속)와 묵직한 베이스 톤으로 정갈한 한국어 독송을 전달합니다. (네트워크 연결이 없는 오프라인 상태에서도 완벽 구동)
   * **온라인 전통 염불:** Everyday Zen Foundation 등 신뢰성 높은 불교 오디오 리소스를 온라인 스트리밍으로 연동하여 실감 나는 전통 합창 염불을 감상할 수 있습니다.
   * **개인 음원 업로드:** 사용자가 가지고 있는 MP3, WAV 등의 불경 음원 파일을 직접 업로드해 목탁 타격과 연계할 수 있습니다.

4. **다이내믹 캔버스 파티클 효과**
   * 목탁을 칠 때 타격점을 중심으로 흩어지는 잔잔한 골드 스파크 입자와 붉은 연꽃잎 파티클이 발생합니다.
   * 마우스를 누르지 않고 있을 때도 화면의 깊이감을 더하는 미세한 백그라운드 앰비언트 먼지 입자가 유영합니다.
   * 타격 시 목탁이 오목하게 눌렸다 펴지는 찌그러짐 물리 애니메이션과 목탁 채가 스윙하는 3D 반동 애니메이션을 탑재했습니다.

5. **수행 기록 및 통계 저장**
   * `localStorage`를 연동하여 오늘 수행한 목탁 타격 누적 횟수와 명상 유지 시간을 실시간 기록하고 저장합니다. (브라우저를 새로고침하거나 종료해도 유지)

## 🚀 로컬 실행 방법

이 프로젝트는 웹 표준 기술(HTML5, CSS3, ES6)로만 개발되었기 때문에, 별도의 빌드 과정 없이 정적 파일 서버를 실행해 바로 구동할 수 있습니다.

### 방법 1. VS Code Live Server 이용 (추천)
1. `D:\Study\WebPage\Moctak` 폴더를 VS Code로 엽니다.
2. 우측 하단의 **Go Live** 버튼을 클릭하여 로컬 서버를 구동합니다.

### 방법 2. Python 내장 서버 이용
명령 프롬프트(CMD) 또는 PowerShell을 열고 아래 명령어를 입력합니다.
```bash
cd D:\Study\WebPage\Moctak
python -m http.server 8000
```
웹 브라우저를 열고 `http://localhost:8000`에 접속합니다.

## 🌐 GitHub Pages (gh-pages) 배포 방법

이 프로젝트는 정적 파일로만 구성되어 있으므로 GitHub Pages에 매우 간편하게 무료로 배포할 수 있습니다.

### 단계별 수동 배포 안내

1. **GitHub 저장소 생성 및 로컬 커밋 설정**
   새로운 깃허브 저장소를 생성합니다. (저장소 이름: `MocTakSori`)
   로컬 폴더에서 Git을 초기화하고 커밋을 진행합니다.
   ```bash
   git init
   git add .
   git commit -m "Initialize Moktak Meditation project"
   ```

2. **원격 저장소 추가 및 기본 브랜치 푸시**
   ```bash
   git remote add origin git@github.com:tramper2/MocTakSori.git
   git branch -M main
   git push -u origin main
   ```

3. **gh-pages 브랜치 배포**
   ```bash
   # gh-pages 브랜치를 생성하고 배포본을 밀어 넣습니다.
   git checkout -b gh-pages
   git push -u origin gh-pages
   
   # 다시 작업 브랜치로 복귀합니다.
   git checkout main
   ```

4. **배포 확인**
   GitHub 저장소의 `Settings` -> `Pages` 탭으로 이동하면 호스팅된 링크를 볼 수 있습니다.  
   주소 형식: `https://tramper2.github.io/MocTakSori/`
